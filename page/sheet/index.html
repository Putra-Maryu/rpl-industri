Shortcut VSC

 Visual Studio Code
    Ctrl + S          : Save
    Ctrl + C          : Copy
    Ctrl + V          : Paste
    Ctrl + Z          : Undo
    Ctrl + Shift + Z  : Redo
    Ctrl + F          : Find
    Ctrl + H          : Replace
    Ctrl + /          : Toggle comment

Shortcut Umum

Ctrl + C → Copy
    Ctrl + X → Cut
    Ctrl + V → Paste
    Ctrl + Z → Undo
    Ctrl + Y → Redo
    Ctrl + A → Select all
    Ctrl + P → Print
    Ctrl + S → Save
    Ctrl + O → Open
    Ctrl + N → New file

Shortcut File Explorer

Ctrl + Shift + N → Buat folder baru
    Alt + Up Arrow → Naik satu folder
    Alt + Left Arrow → Kembali
    Alt + Right Arrow → Maju
    Ctrl + W → Tutup tab File Explorer
    F2 → Rename file/folder
    F3 → Buka pencarian File Explorer
    F4 → Tampilkan address bar
    F5 → Refresh
    Ctrl + Shift + Esc → Buka Task Manager

Shortcut CMD

Command Prompt
    ctrl + C → Copy teks
    Ctrl + V → Paste teks
    Arrow Up → Ulangi perintah sebelumnya
    Ctrl + M → Mode Mark untuk memilih teks
    Ctrl + Shift + Enter → Jalankan 
CMD sebagai Admin

Shortcut Browser

(Chrome, Edge, Firefox)
    Ctrl + T → Tab baru
    Ctrl + W → Tutup tab
    Ctrl + Shift + T → Buka tab yang baru ditutup
    Ctrl + Tab → Pindah ke tab berikutnya
    Ctrl + Shift + Tab → Pindah ke tab sebelumnya
    Ctrl + R / F5 → Refresh halaman
    Ctrl + L / Alt + D → Fokus ke address bar
    Ctrl + F → Cari dalam halaman
